# Period 1

Period 1 is the core library slice plus its first polish pass. It takes vkmtl
from the Vulkan triangle prototype through a working Vulkan + Metal abstraction
and closes with Phase 9 documentation, API, validation, and distribution
cleanup.

These phase notes preserve development decisions from phases 0 through 9. Some
details describe temporary prototype states that have since changed.

Use the main docs as the current source of truth:

- `../roadmap.md` for route and stage boundaries
- `../checklist.md` for active and completed checklists
- `../../api/zh_cn/core.md` and `../../api/en_us/core.md` for the current
  public API
- `../../usage/zh_cn/quick-start.md` and `../../usage/en_us/quick-start.md` for
  current usage
