# Phase 3: Texture Atlas And Material Binding

Phase 3 adds block materials.

## Scope

- Add or generate a small texture atlas.
- Map block IDs to atlas regions.
- Pass atlas coordinates through the chunk vertex format.
- Bind the atlas through normal texture view and sampler APIs.

## Validation

- The example should render visibly different block faces.
- Texture upload should use public vkmtl APIs.
