# Phase 1: Voxel Example Contract

Phase 1 defines the pressure-test target before code starts.

## Scope

- Add `examples/voxel_world` as a public API example.
- Target a block-world renderer prototype, not a full game.
- Require depth testing, indexed chunk meshes, uniform buffers, texture atlas
  sampling, and camera movement.
- Keep physics, inventory, networking, save files, and gameplay out of scope.

## Validation

- The phase should end with a written checklist of vkmtl features the example
  is expected to exercise.
