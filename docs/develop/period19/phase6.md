# Phase 6: Lighting And Visibility Polish

Phase 6 improves visual readability without becoming a full engine.

## Scope

- Add simple directional lighting, vertex lighting, or baked ambient occlusion.
- Keep transparent blocks optional and tightly scoped.
- Preserve stable depth testing and face visibility.

## Validation

- The world should have enough shading contrast to inspect geometry.
- The implementation should remain an example, not a reusable engine module.
